# REST (nodejs express mongo)

## Quick Start

```bash
# Install dependencies
npm install

# Serve on localhost:3000
npm start

# Serve (nodemon) on localhost:3000
npm run dev
```

## API Endpoints

#### Posts Routes

- GET (Protected) /posts
- GET (Protected) /posts/:id
- POST (Protected) /posts
- PUT (Protected) /posts/:id
- DELETE (Protected) /posts/:id

#### User Routes

- POST /user/register
- POST /user/login
